Formulário

Usa-se <form action="" method=""> para criar/dar início a um formulário.

Usa-se <input type="hidden" name="" value=""> para ser um campo escondido, geralmente usa-se isso com id. Serve pra carregar um valor escondido que não quer mostrar pro usuário.

Usa-se <input type="text" name=""> para um campo que aparece a entrada ao digitar.

Usa-se <input type="password" name=""> para um campo que não aparece a entrada ao digitar.

Usa-se <textarea name="" rows="10" cols="30"> para criar uma caixa para poder escrever um texto.

Usa-se várias <input type="radio" name="" value=""> dentro de uma div para que mostre a seleção de opções para que o usuário selecione somente uma opção.

Admin Regular Professor

Usa-se <input type="checkbox" name="" value=""> dentro de uma div para que mostre a seleção de opções para que o usuário selecione as opções.

Ativo

Usa-se várias <option value=""> dentro de um <select name=""> para que mostre a seleção de opções para que o usuário selecione somente uma opção.

Usa-se várias <option value="" multiple size="7">(ou outro número) dentro de um <select name=""> para que mostre a seleção de opções para que o usuário selecione as opções.

Usa-se <button formaction="" formmethod=""> para que, ao clicar envie o formulário. Precisa estar dentro do <form>

Observações

O name="" será a chave do valor que irá pro backend.

O value="" será o valor que irá pro backend.
