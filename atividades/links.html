Links

URL absoluta: para colocar um link e ao clicar para ir em uma página (sairá da atual) usa-se <a href="http://link.com...">. Para abrir em uma nova página usa-se <a href="http://link.com..." target="_blank">. Na URL Absoluta dentro de "href" é obrigatório colocar "http://" caso for indicar um site.

URL relativa: acessa um arquivo .html da sua pasta ou de uma outra pasta de seu projeto. Exemplos: <a href=index.html> → acessa na pasta; <a href=/index.html> → acessa na pasta raiz; <a href=../index.html>, <a href=../../index.html>... → acessa fora da pasta.
