Abrir e Fechar Display: o exemplo acima foi utilizado esse método. Segue abaixo o código e a explicação dele.

Código em JS

document.querySelectorAll('[wm-folder]').forEach(folder => {
folder.onclick = function(e) { //função recebendo um evento
const ul = folder.nextElementSibling
const d = ul.style.display
//visiblidade ul.style.display = d === 'none' ? 'block' : 'none'
}
})

Explicação

Primeiro vai selecionar todos os elementos que está marcado com a propriedade especial "wm-folder" no span, depois irá fazer um forEach pra pegar cada um dos elementos, e quando clicar nesses elementos através do onclick irá disparar uma função que recebe um evento.
const ul = folder.nextElementSibling ➽ vai pegar o elemento depois do span, ou seja, será a ul.
const d = ul.style.display ➽ vai pegar o display da ul, ou seja, todos os seus ﹤li﹥ dentro.
ul.style.display = d === 'none' ? 'block' : 'none' ➽ operação ternária... se o display for estritamente igual a none recebe block para ficar invisível, caso contrário receberá none para ficar invisível.
