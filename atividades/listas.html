Listas

Lista ordenada: usa-se <ol> para criar a lista, e dentro do mesmo usa-se <li> para colocar os itens da lista. Ele irá enumerar os itens da lista.

Aprovados

  1. Robson
  2. Maria
  3. José
  4. Abargededon

Lista não ordenada: usa-se <ul> para criar a lista, e dentro do mesmo usa-se <li> para colocar os itens da lista. Ele irá colocar um ponto preto na frente dos itens da lista.

Lista de Compras

  • Queijo
  • Leite
  • Arroz
  • Pão

Lista de definições: usa-se <dl> para criar a lista, e dentro do mesmo usa-se <dt> para colocar o nome do item e <dd> para definir/digitar algo referenciado ao ultimo <dt>.
