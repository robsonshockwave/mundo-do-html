Código em JS

document.querySelectorAll('[wm-nav]').forEach(link => { //o 'a' é a tag
const conteudo = document.getElementById('conteudo')

link.onclick = function(e) { //evento que recebe do clique
e.preventDefault()
fetch(link.getAttribute('wm-nav')) //chamada ajax
.then(resp => resp.text()) //pega o conteudo html da página que fez uma requisição
.then(html => conteudo.innerHTML = html)
}

Explicação

Pega todos os links e seleciona todas as tags [wm-nav], tem que retornar uma lista de tags, por isso do forEach, ele pegará todos os links de "href wm-nav"... No link.onclick chamará uma função quando o usuário clicar.
e.preventDefault() = vai previnir que a navegação ocorra.
fetch = chamada/requisição ajax do link.
then(resp => resp.text()) = serve para pegar o conteudo html da pagina/requisição do link.
then(html => conteudo.innerHTML = html) = serve para substituir o conteudo da section que tem o id conteudo pelo conteudo do html pego anteriormente.
